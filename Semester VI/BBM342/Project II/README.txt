 \_,| Usage Notes:                                                   | 
    |     -Server must be run in Unix.                               |
    |     -Follow input guidelines.                                  |
    |     -Must download and specifiy Javafx directory.              |

# -*- MakeFile -*-                                                          |
                                                                            |
all: sserver                                                                |
                                                                            |
sserver: sserver.o ; gcc sserver.o -o sserver -lpthread -lrt                |
                                                                            |
sserver.o: sserver.c ; gcc -c sserver.c -std=c99 -g -D_XOPEN_SOURCE=500     |
                                                                            |
clean: ; -rm -f hello                                                       |

-----------------------------------------------------------------------------------------------------------------,
Example command line:                                                                                            |
                                                                                                                 |
./sserver –p <port> –s <channelNumber> –ch1 <file1> –ch2 <file2> –ch3 <file3>                                    |
                                                                                                                 |
java --module-path <path> --add-modules javafx.controls -jar sclient.jar -a <adress> -p <port> -ch <channelNo>   |
                                                                                                                 |

 / \-----------------------------------------------------------------, 
 \_,| Featured:                                                      | 
    |     -Up to 300 clients per channel.                            |
    |     -Dynamic system will interact with you.                    |
    |     -For clients Javafx interface you can use.                 |
    |     -Input checking auto correcting.                           |
    |     -If file is not specified 2-3 channels stream ch1 file.    |
    |     -There is file error, it is covered.                       |
    |     -No memory leaks use one of the following commands to exit.|
    |     -   "exit" or "quit" or "close"                            |
    |     -   "server_exit" or "server_quit" or "server_close"       | 
    |     -When closing a client must try to connect.                |
    |     -FPS can be customized. Default 20.                        |
